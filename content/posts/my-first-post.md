# Welcome!

This is my first blog post built with Hugo and the Terminal theme.

## About This Blog

This blog is:
- Built with [Hugo](https://gohugo.io/)
- Styled with the [Terminal theme](https://github.com/panr/hugo-theme-terminal)
- Deployed to GitHub Pages

## Getting Started

To run this blog locally:

```bash
hugo server -D
```

Then visit `http://localhost:1313/blog/` in your browser.
